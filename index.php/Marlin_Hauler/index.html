Overview

A concept idea which wasn't implemented, the Marlin Hauler as is is a sub-standard M8 with the same model as the Argon Gladiator. It was meant to be the Boron bomber class, but is not available at any shipyard. In an early version of the game, this ship could be captured during station defence missions, but this was patched out. There is a mod which gives this ship the model of the Boron Skate and improves its capabilities.
