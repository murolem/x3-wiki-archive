Usage

this is a modular ship infobox template.

SuppressAutoCat is intended as optional(we'll see how that goes), as such for normal operation, do not include it or leave it blank.

To supress the auto category set as "yes" 

{{Infobox variant
|Variant=
|SuppressAutoCat=
|Faction=
|Class=
|Priceblock=
|Minspeed=
|Maxspeed=
|Minaccel=
|Maxaccel=
|Minsteer=
|Maxsteer=
|Hull=
|Maxshield=
|Shieldslots=
|Maxshieldstrength=
|Shieldreactor=
|Cargoclass=
|Mincargo=
|Maxcargo=
|Lasercap=
|Laserrecharge=
|Gunbay=
|Dockbay=
}}
