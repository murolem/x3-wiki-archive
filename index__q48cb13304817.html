There are 3 versions of Trade Command Software. They permit you to issue trading orders to be carried out automatically.
Trade Command Software is a ship upgrade.

Versions

Available Commands

Trade Command Software Mk1

Permits you to issue the following orders:

  • Buy Ware
  • Sell Ware
      

Trade Command Software Mk2

Permits you to issue the following orders:

  • Buy Ware for Best Price
  • Sell Ware for Best Price
  • Transfer Freight via Drones
      

Trade Command Software Mk3

This version is required for Sector and Universe Traders. Permitted commands are:

  • Start Sector Trader
  • Start Universe Trader


Sector Traders

Main article: Traders
Start Sector Trader is a valid command only if:

  • Trade Command Software MK3
  • Ship type M4, M3, M6, TS, TP, or TM
  • Not personal ship (i.e. you can't be in the ship)

Universe Traders

Main article: Traders
The Universe Trader is only available after certain conditions are met:

  • Trade Command Software MK3
  • Ship type M3, M6, TS, TP, or TM
  • Not personal ship
  • Pilot level 8 or higher (as shown in the pilot name for the ship)

As the pilot gains experience, more sectors can be traversed from the starting point to conduct trades in. Starting sector traders can only trade in the sector they start in.

Availability

The Trade Command Software Mk3 is available at the following docks:

  • Terracorp HQ in Home of Light
  • Teladi Equipment Dock in Ceo's Buckzoid
  • Teladi Equipment Dock in Getsu Fune
  • Teladi Equipment Dock in Seizewell
  • Teladi Equipment Dock in The Vault
  • Teladi Equipment Dock in PTNI Headquarters
  • Teladi Equipment Dock in Two Grand
