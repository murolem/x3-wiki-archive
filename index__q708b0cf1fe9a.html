A ship's autopilot allows the ship to perform automated tasks. These tasks are defined based upon installed command software and ship's class.

[edit] Software Types

Allowed Options

[edit] Explorer Command Software

  • Deploy Satellite Network
  • Fly Through Gate...
  • Map Sector...
  • Scan Asteroids

[edit] Fight Command Software MK1

  • Attack all enemies
  • Attack Capital Ships
  • Attack Fighters
  • Attack...
  • Protect...
  • Turret Attack Enemies
  • Turret Protect Ship
     

[edit] Fight Command Software MK2

  • Attack all enemies, land
  • Attack enemy and land...
  • Attack my target
  • Attack nearest enemy of...
  • Attack shields of...
  • Attack target of...
  • Board Ship
  • Defend Position
  • Defend Sector...
  • Defend Station
  • Launch All Marines...
  • Missile Barrage...
  • Retreat
  • Turret Missile Defense
  • Turret: Attack Capitals
  • Turret: Attack Fighters
  • Turret: Missiles Only

[edit] Navigation Command Software

  • Jump and Fly to Pos
  • Jump and Fly to Station

[edit] Patrol Command Software

  • Patrol Sector
  • Patrol Sectors
  • Scout for Enemies

[edit] Supply Command Software

  • Equip Ships Like...
  • Rearm Ships
  • Refuel Ships
  • Resupply Ships

[edit] Trade Command Software Mk1

Permits you to issue the following orders:

  • Buy Ware
  • Sell Ware
     

[edit] Trade Command Software Mk2

Permits you to issue the following orders:

  • Buy Ware for Best Price
  • Sell Ware for Best Price
  • Transfer Freight via Drones
     

[edit] Trade Command Software Mk3

This version is required for Sector and Universe Traders. Permitted commands are:

  • Start Sector Trader
  • Start Universe Trader
