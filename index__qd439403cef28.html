Description

Encyclopedia Entry

This bleak region is home to the Yaki faction. Little else is known about this sector.

Information

There is an ownerless Advanced Eclipse in this sector in Terran Conflict.

Default Stations

Ocracoke's Storm is empty by default. If you are friendly with the Yaki, it makes a good sector for player complexes.
